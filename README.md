# image-lat-long

> Relied on Node JS server

- Check out [live app](https://image-lat-long.herokuapp.com/)

## To run this app

1.  Clone this repository
2.  Move in to the folder with this 
    ```bash 
          cd  image-lat-long 
      ```
3.  run npm install

```bash
npm i
```

4. run node server

```bash
node server.js
```
